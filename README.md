## 适用对象

面向开发者/创客市场及教育市场（高校、K12、政府示范项目）的AI硬件公司，尤其是：

- 拥有自研边缘AI硬件产品
- 海外运营从零起步
- 希望借鉴 M5Stack、DFRobot、Seeed Studio 等同行的成功路径

## 核心结论

1. **封装即品牌** — 同芯片不同封装 = 品牌差异化
2. **合作伙伴出海** — 初期优于自建海外办事处
3. **SDK开源 / 模型闭源** — AI硬件时代的开源策略
4. **众筹优先** — 零稀释 + 市场验证 + 种子用户
5. **统一SDK锁定生态** — 一套API覆盖所有硬件
